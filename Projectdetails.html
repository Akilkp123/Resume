Project Patient Accounting Masterfiles
Client Name Hospitals Corporation Of America
Company Atos|Syntel
Technologies C#.Net, MVC, Html, JQuery, JavaScript , CSS, DB2

Project Profile

This application is developed to manage all the collection of records related to a facility. Facility is nothing but hospitals, healthcare centers etc. It manages the data related to all the departments like Billing, Insurance, and Customers etc. related to a particular facility. Also have an export and import facility for bulk data upload and retrieval.
